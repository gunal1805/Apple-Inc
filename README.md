# Student Performance Prediction

## 📌 Overview
This project predicts whether a student will pass or fail using machine learning techniques.

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## 📊 Model Used
- Logistic Regression
- Decision Tree

## 📈 Outcome
- Successfully built a classification model with good accuracy
